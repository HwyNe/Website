// 02 PicoCTF Deneyimim

🎯 PicoCTF

43 Web Challenges Solved

PicoCTF, CTF dünyasına giriş yapmak için harika bir platform. Özellikle web exploitation kategorisinde çok şey öğrendim. 43 web sorusunu çözerken her biri farklı bir teknik ve yaklaşım gerektirdi.

Öğrendiğim Teknikler

💉 SQL Injection

Blind SQLi, Union-based, Error-based techniques

🎭 XSS (Cross-Site Scripting)

Reflected, Stored, DOM-based XSS

🔐 Authentication Bypass

Session hijacking, JWT manipulation

📂 Path Traversal

LFI/RFI, Directory traversal

🔄 SSRF

Server-Side Request Forgery

🎪 IDOR

Insecure Direct Object References

En Sevdiğim Challengelar

  • SQL Injection Advanced: Blind SQLi ile database dump etmek çok eğlenceliydi
  • XSS Filter Bypass: WAF bypass teknikleri öğrenmek harikaydi
  • JWT Manipulation: Token'ları manipüle edip admin olmak
